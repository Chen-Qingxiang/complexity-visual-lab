Deterministic chaos

Logistic Map Explorer

The logistic map is defined by xt+1 = r xt (1 - xt). It is deterministic: the next value is fully determined by the current value and r. As r increases, the system can move from a stable fixed point to periodic cycles and then to chaos.

Controls

Adjust the parameter, initial condition, and iteration limits. All plots update together.

Time Series

The sequence can settle to one value, alternate through cycles, or behave irregularly.

Cobweb Diagram

The path moves vertically to the curve and horizontally to the diagonal, showing each iteration.

Bifurcation Diagram

Long-run values reveal period-doubling and chaotic regions as r varies.

Sensitivity

In chaotic regions, two very close initial values can diverge quickly. This illustrates why deterministic systems can still be difficult to predict long-term.
