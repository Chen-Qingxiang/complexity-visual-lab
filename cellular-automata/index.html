What this shows

Elementary automata

An elementary cellular automaton is a row of cells that can each be off or on. A new row is computed from every three-cell neighborhood in the previous row: left, center, and right.

Rule numbers

The rule number is an 8-bit lookup table. Its bits define the output for neighborhoods 111, 110, 101, 100, 011, 010, 001, and 000, in that order.

Notable rules

Rule 30 creates chaotic, random-looking patterns. Rule 90 draws the Sierpinski triangle. Rule 110 supports complex moving structures and is computationally universal. Rule 184 models particle-like traffic flow.
